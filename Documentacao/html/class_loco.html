Descrição detalhada

Classe para criar um Loco a partir de 2 Alelos.

A Classe Loco tem como funcao armazenar o genotipo (configuracao alelica) de cada loco que ira compor um individuo, nao importando o tipo de loco (marcador, QTL ou poligenico).

Metodo para ajustar o vetor static de contribuicao dos locos poligenicos.

Parâmetros:
varad- Variancia Aditiva fornecida pelo usuario;
qtdlocos- Quantidade de Locos Poligenicos que irao formar o genoma do individuo;
qtdqtl- Quantidade de Locos QTLs que irao formar o genoma do individuo;
qtdmarcador- Quantidade de Locos Marcadores no genoma de cada individuo;
freqp- Frequencia alelica (p) dos locos poligenicos (atribui-se a mesma frequencia para todos os locos poligenicos);
pvaqtl- Proporcao da variancia aditiva que eh explicada pelos QTLs.
