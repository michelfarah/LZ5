Descrição detalhada

Classe para criar um Alelo.

A classe Alelo tem como funcao representar um alelo na forma mais simples, podendo ser 0 ou 1, esta classe apenas representa o estado do alelo, sem armazenar valores ou efeitos para este. Esta eh uma classe base e, dependendo do tipo de simulacao pode-se usar a classe derivada AleloId.
