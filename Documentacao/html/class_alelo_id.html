Descrição detalhada

Classe derivada para criar um Alelo com sua identificacao (ID).

A classe AleloId eh uma classe derivada da Alelo e tem como finalidade criar um alelo marcador com identificao para estudos que envolvem o conceito de locos identicos por descendencia (ibd). Assim, quando desejado, o simulador cria uma identificacao unica para cada novo Alelo criado.
