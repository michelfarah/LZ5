LZ5 by Lucca-z

LZ5 is a software for data simulation to be used mainly in animal breeding area. LZ5 allows the user to compare methodologies, to create scenarios to illustrate scientific articles and even to avoid costs relating to field experiments.

LZ5 is a software which is divided in 4 logical modules comprising correlated function and procedures: 1. Data Simulation 2. User Interface 3. Report 4. Genetic Evaluation LZ5 is a Free-Software in C++ for the Linux plataform

Dependencies

Linux and GSL (GNU Scientific Library)

Install

The installation procedure is to copy all the files to the directory you intend to run the program.
